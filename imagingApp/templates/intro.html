Introduction to Frequency Analysis

Frequency domain analysis is essential to the field of image processing as it greatly simplifies the computational cost of applying filters. In the example of a simple low pass filter, a simple spatial blurring kernel must be convolved over all pixel indices and multiple mathematical operations must be performed to compute the transformed value of each pixel. However, if the same image is first transformed into a frequency domain, applying a low pass filter requires just a single multiplication for each point. The resulting frequency matrix can then be inverse Fourier transformed to unveil the blurred image. While the process of Fourier and inverse-Fourier transforms might first seem intimidating or complicated to many new learners, this process is incredibly fast with current computer science algorithms, much faster than spatial filters.
It is essential for undergraduates is electrical engineering, computer science, and related fields to gain an intuitive understanding of how these filters operate in order to expertly apply them through courses in image analysis. However, as stated earlier, this understanding is not always so straightforward. For this reason, this online interactive website was created using a Django framework as well as opencv and matplotlib libraries to provide both information and interactive tutorials for new learners. This work builds from static tutorials such as HIPR2, a project created by the Department of Artificial Intelligence in the University of Edinburgh.
